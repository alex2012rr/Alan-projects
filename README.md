<h3>Teoria de html</h3>
<p>1. El Concepto de Marcado (Tags)
La idea central del HTML es envolver pedazos de texto con etiquetas para darles una función específica. Estas etiquetas le dicen al navegador: "esto no es solo texto, es un título", o "esto es un enlace".
Anatomía de un Elemento
Un elemento HTML se compone generalmente de tres partes:
Etiqueta de apertura: <etiqueta> (Indica dónde empieza el efecto).
Contenido: El texto o imágenes dentro.
Etiqueta de cierre: </etiqueta> (Indica dónde termina).</p>
